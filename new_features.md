# 更新计划
## 忽略大小写（如果没有实现则实现，实现了则忽略此条）
匹配网页上单词的时候，应该忽略大小写。



## 导入和本地单词合并
在扩展选项页，目前已经实现了导入 saladict的JSON文件（以下称“外部单词”）。
- 另加一个输入框 "用户单词区"，用户可以自己输入单词，一行一个，此框的单词和外部单词合并，最终合并后的单词表作为最终单词表进行匹配，并整体存入 Storage
- 外部单词的显示框背景变灰，以为不可编辑之意
- 合并两表单词时注意去重

### 整体导入导出
- 按钮名称就是 “导入” “导出”，将目前saladitct 的“选择文件”“保存” 、外部单词显示区，放到单独的区域内以作区分
- 新加【整体导入导出】功能，格式为JSON，基本就是 storage 的导出。
- 整体导入的单词放在"用户单词区"。如果整体导入时"用户单词区"有单词，则合并后覆盖，并用 toast 提示本次导入的单词数量、合并的单词数量等等信息。
