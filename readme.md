# 生词高亮显示插件 (Vocabulary Highlighter)

一款 Chrome 浏览器扩展，旨在通过在您访问的任何网页上自动高亮词汇表中的单词，来帮助您学习英语。

## 功能

- **自动高亮**: 词汇表中的单词会在您访问的网页上自动高亮显示。
- **词汇管理**: 通过扩展的选项页面，轻松管理您的单词列表。
- **从 Saladict 导入**: 支持从“沙拉查词 (Saladict)”的 JSON 文件导入单词列表。
- **用户自定义单词**: 您可以手动添加和管理自己的单词列表。
- **合并词汇表**: 导入的单词和用户自定义的单词会自动合并，并去除重复项，形成一个最终用于高亮显示的词汇表。
- **整体导入/导出**: 支持以 JSON 格式备份和恢复您的整个词汇表。
- **边框模式**: 您可以选择使用背景高亮，或仅用边框来标记单词，以获得更简洁的视觉效果。

## 安装

1. 下载本项目的所有文件。
2. 打开 Chrome 浏览器并访问 `chrome://extensions/`。
3. 在页面右上角，启用“开发者模式”。
4. 点击“加载已解压的扩展程序”，然后选择本项目的文件夹。
